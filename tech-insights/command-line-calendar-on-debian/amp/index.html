Command line Calendar utilities on Debian

3 weeks ago

Linux distributions provides simple and useful utilities through command line. Calendar in GNU/Linux might seem like novelty, however mastering the tool may be helpful.

There are many command line interface (CLI) utilities for calendar.

First CLI utility is cal command, cal command is available in most distributions. cal command displays current month by default with highlight on present day. It is included with bsdmainutils package which can be installed by using the command –

apt-get install bsdmainutils

The simple cal command gives output of present month calendar with highlighted day.

$ cal

There is also a command ncal, which is is included as well but offers an alternative layout.

$ ncal

What is the difference between cal and ncal ?

While cal presents a week horizontally and ncal display a week in calendar as vertical, there is much more to it as well. cal command was present in AT&T UNIX and ncal was used in FreeBSD. To keep compatibility with older Unix’s cal command, it would output 8 lines, as cal command was used in many CGI scripts.

Lets see some examples :

To Print for particular month and year
cal (month) (year)

$ cal 6 2019

To Print the previous, current and next month surrounding today

$ cal -3

To Print for a specific month
cal -m (month)

$ cal -m 2

Remember :

An unnoticed behaviour by user may lead to altering result in cal command, how you may ask?

If you were to display a calendar of month June of 2018, you may enter

cal 6 2019

which is correct, however if you type,

cal 6 19

it will display June from year 19 AD, so remember to READ THE MANUAL.

Try cal 9 1752, Skipped 11 Days in calendar?

yes, that is due to Gregorian Reformation.

Now lets try different utility, gcal

It can be install with –

apt-get install gcal

gcal is GNU utility, it can calculate astronomical data, displays eternal holiday lists for countries around the world, fixed date lists for reminders and many more.

To print current month of year

gcal

To print previous, current and next month surrounding today

gcal .

To print month of year

gcal (month) (year)

gcal 6 2019
