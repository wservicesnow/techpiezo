Install WordPress on Windows 10 using XAMPP

10 months ago

WordPress is a Content Management System for building websites. In this article, we would discuss how to install & configure WordPress using XAMPP.

All you have to do is install XAMPP for Windows through a .exe file. Afterwards, you have to download and configure WordPress. We will discuss these steps in detail next.

How to Install WordPress on Windows 10 using XAMPP?

STEP – I: Download XAMPP for Windows from the Apache Friends official website. Head to Downloads section to download the relevant application.

The file size is around 144 MBs and download time would depend on your Internet connection speed. There are multiple versions available, download one of your choice. After the file gets downloaded, you have to run the setup. Default installation folder is C:\xampp\

STEP – II: Download WordPress:

Download the relevant package from official website with .zip extension. Once, the file gets downloaded. Extract it to a desired folder either with the help of software application or through in-built Windows support by right-clicking it.

When the file gets extracted, you have to copy the entire extracted directory to the location C:\xampp\htdocs\

STEP – III: Now, we have to enable Apache, MySQL services from the XAMPP control panel to configure WordPress. Open the XAMPP control panel. You have to start Services modules for Apache and MySQL.

Subsequently, you have to cross-check if your XAMPP installation is working as required. You have to open your web browser and type http://localhost in address bar. If XAMPP is installed and running successfully, your web browser will return with Welcome message from XAMPP.

Next, click Admin button available next to MySQL service stop module. This is being done to create a database for our WordPress installation. After clicking the Admin button you would be redirected to web page for phpMyAdmin. Click on database tab and create a database with the name that would be provided when we configure WordPress.

Now, open C:\xampp\htdocs\wordpress\ folder and copy wp-config-sample.php to wp-config.php in the same folder. Now, you have to locate entries provided below and make suitable changes in the wp-config.php file.

/** The name of the database for WordPress */
define( 'DB_NAME', 'wordpress' );

/** MySQL database username */
define( 'DB_USER', 'root' );

/** MySQL database password */
define( 'DB_PASSWORD', '' );

/** MySQL hostname */
define( 'DB_HOST', 'localhost' );

Thereafter, we would proceed with WordPress installation. Visit

http://localhost/wordpress/

in your web browser. It will take you to the installation page. In order to login to WordPress, you will be asked to provide username and password.
